Özel üretim arama sayfasına hoşgeldiniz! Bu demo sayesinde, örnek özel üretim verileri üzerinde dilediğiniz kriterlere göre arama yapabilirsiniz. Demo üzerindeki alanlar arttırılabilir ve geliştirilebilir. Veriler halihazırda XML ortamında saklanmaktadır, ancak gerekirse Excel, veritabanı, ERP sistemi, vb farklı veri kaynakları da kullanılabilir.

Kullanıcı arayüzü HTML5 + CSS3 + JavaScript ile, sunucu tarafındaki arama motoru ise Java ile geliştirilmiştir.
